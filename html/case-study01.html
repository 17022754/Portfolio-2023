Laat ElineSpreekt spreken

UX / UI design - 2022

ElineSpreekt is een applicatie die mensen met een beperking helpt communiceren. De website moet een hulpmiddel zijn voor begeleiders, zodat zij ook begrijpen hoe de applicatie gebruikt kan worden en hoe zij de applicatie kunnen aanschaffen.

Probleem

De website van ElineSpreekt is verouderd, waardoor de ontwikkelingen van de applicatie niet zichtbaar zijn. Hierdoor sluit de informatie die op de website staat, niet meer aan bij de app. Hierdoor sluit de website niet meer aan bij de primaire doelgroep en bij de identiteit van ElineSpreekt, ‘makkelijk communiceren voor iedereen’. Het is niet duidelijk hoe de informatie beter kan worden overgebracht op de primaire doelgroep.

Doel

Het doel van het project is om de website van ElineSpreekt te verbeteren op functionali- teit, gebruiksvriendelijkheid en efficiëntie, door de informatievoorziening op de website te verbeteren.

Met dit project heb ik gebruik gemaakt van de ‘double diamond’ methode. Deze methode bestaat uit vier fases en na elke fase worden alle bevindingen getest. In de eerst fases heb ik een usability test en customer journey uitgevoerd, om de grootste pijnpunten van de gebruikers vast te leggen.

Door de bevindingen konden de wireframes worden opgezet. Deze zijn ook getest met de gebruikers en deze feedback is verwerkt in het uiteindelijke resultaat. Het resultaat is een prototype van de website.
