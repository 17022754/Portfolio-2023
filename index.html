Leergierig, vol passie & ondernemend


Sinds ik bezig ben met (UX) design heb ik mijn passie gevonden. Elk project leer ik iets nieuws en daag ik mijzelf uit om iets nieuws te leren. Op deze manier heb ik leren animeren, content voor instagram leren maken en maak ik ondertussen bijna dagelijks gebruik van Figma.

Tijdens mijn stageperiode heb ik bij DutchGiraffe gewerkt, waar ik heb geleerd hoe ik magazines maak en hoe ik met Wordpress werk. Na mijn stageperiode mocht ik ook blijven werken, waardoor ik mijn designsskills verder kon ontwikkelen.

Naast mijn studie ben ik graag actief, zowel binnen mijn studievereniging als voor de introductieweek. Door het draaien van een aantal bestuursjaren heb ik een duidelijk beeld van hoe je een goede communicatie tot stand brengt. Dit kan ik inzetten tijdens mijn projecten, omdat communicatie een groot onderdeel is van designen.
